Ноутбуки

September 3, 2025

Типи ноутбуків

  • Бюджетні ноутбуки – дешевші моделі для серфінгу, офісних програм, перегляду відео. Для кого: студенти, базові користувачі.
  • Ультрабуки – тонкі, легкі, з довгим часом роботи, SSD, потужні процесори. Для кого: подорожуючі, студенти, ті, хто працює в дорозі.
  • Ноутбуки для бізнесу – підвищена надійність, функції безпеки, великі екрани. Для кого: підприємці, ІТ-фахівці, корпоративні користувачі.
  • Геймерські ноутбуки – потужні відеокарти, FPS, охолодження, екрани 120–240 Гц. Для кого: геймери, стрімери, 3D-дизайнери.
  • Для творчих професій – високоточні екрани, великі обсяги пам’яті, потужні GPU. Для кого: дизайнери, фотографи, відеоредактори, музиканти.

Типи екранів

  • TN – швидкий відгук (1–3 мс), високі частоти 120–240 Гц, дешеві. Недоліки: погані кути огляду, слабка передача кольорів. Для: геймерів, бюджетних користувачів.
  • IPS – широкі кути огляду (178°), точні кольори, яскравість. Недоліки: дорожчі, IPS Glow. Для: дизайнерів, фотографів, мультимедіа.
  • VA – високий контраст, кращі кольори, ніж TN. Недоліки: довший відгук, не така точність як IPS. Для: кіно, текстів, середніх геймерів.
  • OLED – ідеальний чорний, безмежна контрастність, миттєвий відгук. Недоліки: висока ціна, ризик вигорання. Для: фотографів, геймерів, любителів якісної картинки.
  • LED – яскравість, енергоефективність, тонкі. Недоліки: час відгуку повільніший, ніж OLED. Для: універсальних користувачів.

Процесор (CPU)

  • Кількість ядер: більше ядер – більше багатозадачності.
  • Частота: вимірюється в ГГц, визначає швидкість виконання інструкцій.
  • Техпроцес: менший нм = енергоефективність і менше нагрівання.
  • Hyper-Threading/SMT: ядро виконує 2 потоки.
  • TDP: визначає тепловиділення.

Основні моделі

  • Pentium / Celeron: для простих завдань.
  • Core i3 / Ryzen 3: серфінг, документи, відео.
  • Core i5 / Ryzen 5: офіс, мультимедіа, легкі ігри.
  • Core i7 / Ryzen 7: відео, 3D, ігри середнього рівня.
  • Core i9 / Ryzen 9: професіонали, рендеринг, моделювання.

Типи процесорів

  • U (Ultra-low Power): для ультрабуків, 15 Вт TDP.
  • H (High Performance): для ігрових ноутбуків, 45 Вт+.
  • HX (Extreme Performance): топові робочі станції, 55 Вт.

Це "мозок" ноутбука, відповідає за виконання програм, швидкість обробки даних.

Відеокарта (GPU)

  • Інтегрована: енергоефективна, бюджетна, але слабка. Для: офіс, серфінг, відео.
  • Дискретна: окрема з VRAM, потужна для ігор і графіки. Для: ігри, рендеринг, дизайн.

Основні класи

  • Бюджетний: GTX 1650, RX 5500M – легкі ігри.
  • Середній: RTX 3050, RX 5700M – ігри, монтаж.
  • Високий: RTX 3070/3080, RX 6800M – AAA-ігри, 3D.
  • Топовий: RTX 4080/4090 – проф. графіка, рендеринг.

Оперативна пам’ять (RAM)

  • 8–16 ГБ: офіс, прості ігри.
  • 16 ГБ: ігри на високих налаштуваннях, монтаж.
  • 32 ГБ+: рендеринг, віртуалізація.

Вбудована пам’ять

  • HDD: великі обсяги, але повільні.
  • SSD: швидкі, безшумні, дорожчі.
  • NVMe SSD: ультрашвидкі (до 7 ГБ/с), дорогі.
  • eMMC: дешеві, повільні, для бюджетних.

Порти та підключення

  • USB-A: миші, клавіатури, флешки.
  • USB-C: зарядка, передача даних.
  • HDMI: монітори, телевізори.
  • Thunderbolt 3/4: 40 Гбіт/с, зовнішні GPU.
  • 3.5 мм: навушники, мікрофони.
  • Ethernet: стабільний інтернет.

Додаткові особливості

  • Тачпад з мультитач жестами.
  • Підсвічування клавіатури / RGB.
  • Сенсорний екран, 2-в-1 трансформери.
  • Бездротова зарядка (Qi).
  • Розпізнавання обличчя, сканер відбитків.
  • Wi-Fi 6, FullHD вебкамера.
  • Системи охолодження з 2 вентиляторами.
  • 120 Гц+ дисплеї, низький час відгуку (1–3 мс).
  • Покращені дискретні відеокарти.
  • Механічні клавіші.
